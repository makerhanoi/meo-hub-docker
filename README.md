To start first time, you can run with a command below
```
docker-compose up -d
```

To install meo-esp, there are two ways:
1. In your shell
```
cd node-red\data
npm -i node-red-contrib-meo-iot
npm -i node-red/node-red-dashboard
cd ..\..
docker-compose restart
```
